# Swans Front-End

**Как клонировать?**
```
git clone https://github.com/lebedev05tmn/swans-fe.git
cd swans-fe
```
**Как запушить, пока нет ветки dev?**
```
git add .
git commit -m "layout: example fitch"
git push origin main
```
